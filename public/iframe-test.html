ChatKit Iframe Test

Testing: chatkit-iframe.html

This page demonstrates how to embed the ChatKit widget in an iframe.

Auto-parameter forwarding: Any URL parameters on this page (e.g., ?firstname=Toni&email=toni@example.com) are automatically passed to the iframe!

Try it: ?firstname=John&email=john@example.com

Embedding Code:

<iframe 
  src="/chatkit-iframe.html?firstname=John" 
  width="100%" 
  height="600" 
  frameborder="0"
  style="border: none;"
  allow="clipboard-read; clipboard-write">
</iframe>
