Assistant Chat Embed Examples

Note: Replace YOUR_DOMAIN with your actual domain (e.g., https://your-app.vercel.app or https://your-domain.com)

Example 1: Basic Embed

<iframe
    src="YOUR_DOMAIN/assistant"
    width="100%"
    height="800"
    frameborder="0"
    allow="clipboard-read; clipboard-write"
    title="FYI Support Assistant"
></iframe>

Example 2: With Personalized Greeting (first-name parameter)

<iframe
    src="YOUR_DOMAIN/assistant?first-name=John"
    width="100%"
    height="800"
    frameborder="0"
    allow="clipboard-read; clipboard-write"
    title="FYI Support Assistant"
></iframe>

Example 3: Responsive Embed with CSS

<style>
.assistant-container {
    position: relative;
    width: 100%;
    padding-bottom: 75%; /* 4:3 aspect ratio */
    height: 0;
    overflow: hidden;
    border-radius: 8px;
    border: 1px solid #e0e0e0;
}
.assistant-container iframe {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    border: none;
}
</style>

<div class="assistant-container">
    <iframe
        src="YOUR_DOMAIN/assistant?first-name=Jane"
        allow="clipboard-read; clipboard-write"
        title="FYI Support Assistant"
    ></iframe>
</div>

Example 4: Fixed Height (Recommended for Chat)

<iframe
    src="YOUR_DOMAIN/assistant?first-name=Alex"
    width="100%"
    height="700"
    frameborder="0"
    allow="clipboard-read; clipboard-write"
    title="FYI Support Assistant"
    style="border-radius: 8px; border: 1px solid #e0e0e0;"
></iframe>

Usage Instructions

  • Basic URL: YOUR_DOMAIN/assistant
  • With first name: YOUR_DOMAIN/assistant?first-name=John
  • Alternative parameter: YOUR_DOMAIN/assistant?firstName=John (also supported)
  • Height: Recommended height is 600-800px for optimal chat experience
  • Width: Use 100% for responsive design, or fixed width like 800px
